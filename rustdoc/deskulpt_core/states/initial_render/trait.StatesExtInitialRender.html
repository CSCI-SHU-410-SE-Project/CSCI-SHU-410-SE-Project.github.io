fn set_render_ready(&self) -> Result<()>

Set the canvas as ready to render widgets.

If there is a pending payload, a render-widgets event will be emitted to the canvas with that payload.

Source

fn emit_on_render_ready(&self, payload: Value) -> Result<()>

Emit a render-widgets event to the canvas when it is ready.

If the canvas is already ready to render widgets, emit the given payload to the canvas immediately. Otherwise, store the payload as pending so that it can be emitted later when the canvas is ready.
